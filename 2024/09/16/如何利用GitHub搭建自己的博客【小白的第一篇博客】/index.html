如何利用GitHub搭建自己的博客【小白的第一篇博客】

hi，大家好，我是小白，这是小白的第一篇博客！

博客名称是：如何利用GitHub搭建自己的博客

我是根据知乎文章的步骤搭建的：【保姆级】利用Github搭建自己的个人博客，看完就会 - 知乎 (zhihu.com)

非常详细！感谢大佬！！😀

下面记录一下我在这个过程中遇到的错误吧。

1、使用命令 npm install hexo -g 安装hexo的时候报错，报错信息如下

尝试了使用管理员打开cmd和设置user具有文件夹的所有权限，没有用，需要新建两个文件夹node_global和node_cache，使用命令指定一下node的prefix和cache的路径

然后在环境变量中的用户变量新加上这一项

再运行 npm install hexo -g 就可以了。

【参考博客：nodejs安装、更改全局位置_node全局位置-CSDN博客】

2、运行 npm install hexo -g 成功，但是运行 hexo -v 报错，报错信息如下

需要找到hexo安装目录，将.bin路径添加到系统变量中

如果依然报错hexo: command not found，重启电脑。

【参考博客：解决”bash: hexo: command not found”问题-CSDN博客】

3、知乎教程中访问xxx.github.io时，报错404

需要把仓库名xxx.github.io中的xxx改成 自己注册GitHub账户时的用户名

在这里rename即可
